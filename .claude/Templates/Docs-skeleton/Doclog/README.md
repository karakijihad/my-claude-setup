# Doclog Index

Architecture decisions, one file per day. Newest date at the top. Multiple decisions in a single day share one file, newest timestamp first within.

| Date | File | Titles |
|-|-|-|
| YYYY-MM-DD | [YYYY-MM-DD.md](./YYYY-MM-DD.md) | [titles of decisions recorded that day] |

Format per file: see `~/.claude/Templates/doclog-entry.md`.

**An audit is not a decision.** Audits live in `../Audit/`. Accepted decisions live here.
