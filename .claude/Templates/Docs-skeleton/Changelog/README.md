# Changelog Index

Semver changelog, one file per day. Newest date at the top. Multiple changelog entries in a single day share one file, newest timestamp first within.

| Date | File | Versions / notes |
|-|-|-|
| YYYY-MM-DD | [YYYY-MM-DD.md](./YYYY-MM-DD.md) | [e.g., 1.2.0 release, hotfix] |

Format per file follows Keep a Changelog (see `~/.claude/Templates/changelog-entry.md`).
